## Concept-Area Map

The 38 red-dot topics group naturally into 7 concept areas. The flashcard files mirror this grouping.

1. **Software Engineering Foundations** — what software engineering is, sustainable code
2. **Testing** — TDD, definition of testing, test pyramid, levels (unit/integration/system/acceptance), black-box vs white-box, designing for testability, V-model journey
3. **Design by Contract** — formal contracts, preconditions/postconditions/invariants
4. **Software Maintenance** — definition, maintainer's role, maintainability, estimation, planning levels, refactoring, reverse engineering, software archaeology
5. **Software Quality** — quality principles, software quality definition, CoSQ, SQM processes, iron triangle, V&V
6. **Software Configuration Management (SCM)** — definition, four core functions, SCM↔SQA, identification, SCIs, baselines
7. **Software Engineering Management** — what SE management is, three pillars (People/Product/Process), emotional intelligence, communication directions, one-on-ones
